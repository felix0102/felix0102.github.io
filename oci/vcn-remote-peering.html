# Osaka <--> Jaddah

  • Osaka vm 10.1.1.2 VCN-A
  • Jaddah vm 10.3.1.2 VCN-C

# build connection step

  1. In Region Osaka
  2. Create DRG-1
  3. Create Remote Peering-1
  4. Attach VCN-A
  5. Add DRG Route 10.3.0.0/16
  6. In Region Jaddah
  7. Create DRG-2
  8. Create Remote Peering-2
  9. Attach VCN-C
  10. Add DRG Route 10.1.0.0/16
  11. Establish remote peering connection from Osaka to Jaddah
